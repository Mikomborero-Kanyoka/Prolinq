# Prolinq Advertisement Template Generator

## Usage Instructions
Replace the placeholder variables below with your specific item details to generate clean, modern advertisements suitable for the Prolinq platform.

---

## INPUT DATA TEMPLATE

```

Name: {{ITEM_NAME}}
Category: {{CATEGORY}}
Company Name: {{COMPANY_NAME}}
Price/Rate (optional): {{PRICE}}
Main Benefit / Value: {{BENEFIT}}
CTA Button Text: {{CTA_TEXT}}
```

## DESIGN RULES COMPLIANCE CHECKLIST

### Image Ad Rules:
✅ Minimalist, modern, mobile-friendly layout
✅ Clean background with soft gradients
✅ Clear visual focus that represents the item
✅ Short text inside the image (max 6–7 words)
✅ No clutter, no tiny text

### Text Ad Rules:
✅ Headline max 6 words
✅ 1–2 line description
✅ Optional offer/price
✅ CTA max 3 words
✅ No emojis
✅ Professional, clean, easy to read

---

## CUSTOMIZATION TIPS

### For Services:
- Focus on expertise and results
- Include credentials or experience
- Emphasize time-saving or problem-solving benefits

### For Products:
- Highlight quality and features
- Include warranty or guarantee information
- Focus on practical benefits

### For Events:
- Create urgency with dates/time
- Highlight networking or learning opportunities
- Include speaker credentials if applicable

### For Gigs:
- Emphasize quick turnaround
- Show portfolio or past work
- Include clear deliverables

---

## BRAND CONSISTENCY FOR PROLINQ

### Color Palette:
- Primary: Blue gradients for technology/services
- Secondary: Purple for creative/marketing items
- Accent: Orange for events/time-sensitive offers
- Neutral: Green for products/physical items

### Typography:
- Headlines: Bold, sans-serif
- Body text: Clean, readable sans-serif
- Image text: High contrast, large enough for mobile

### Visual Style:
- Minimalist approach
- Plenty of white space
- Professional photography or illustrations
- Consistent gradient backgrounds
- Mobile-first design approach

---

This template system ensures all Prolinq advertisements maintain brand consistency while effectively communicating the value proposition of any item, service, or event on the platform.
