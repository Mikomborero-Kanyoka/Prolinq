PERFORMANCE CONSIDERATIONS
================================================================================

Load Time Factors:
- Database size (more jobs = slightly longer queries)
- Network latency to backend
- Chart rendering time (minimal with Recharts)
- Typical: 200-500ms total load time

Optimization Tips:
- Data is not cached - fresh on each visit
- Consider adding Redis cache for large databases
- API response is typically <1MB
- Charts render efficiently with Recharts

Database Impact:
- Queries use proper indexing (existing columns)
- Extraction-based calculations (no heavy joins)
- No N+1 query problems
- Suitable for databases with 100k+ jobs
